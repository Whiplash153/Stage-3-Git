# Markdown — Code & Blocks

## Purpose
This task introduces the Markdown features used to format code in technical
documentation. These elements are essential for README files, tutorials,
and developer guides.

## Covered topics
- Inline code using single backticks
- Fenced code blocks using triple backticks
- Syntax highlighting for Python examples
- Blockquotes
- Nested code blocks inside blockquotes
- Mixed content (paragraphs, lists, inline code, and multi-line blocks)

## Files
- `example.md` — contains all practice examples
- `task.py` — summary of the work completed
- `README.md` — explanation of the task
