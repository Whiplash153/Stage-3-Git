## Inline code
Use backticks to highlight short code snippets within text.

Example: `print("Hello, world!")`

---

## Code blocks
Triple backticks create a multi-line code block:

```
print("Line 1")
print("Line 2")
```

---

## Syntax highlighting
You can specify a language for colored formatting:

```python
def greet(name):
    return f"Hello, {name}!"
```
---
## Nested elements

You can nest code blocks inside quotes:

> Example:
> ```python
> for i in range(3):
>   print(i)
> ```

---
## Mixed content

Paragraph + inline code + list + block:
