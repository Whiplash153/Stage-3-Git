# Markdown — Images

## Purpose
This task demonstrates how to work with images in Markdown documents.
Images are often used in README files, documentation, tutorials, and reports.

## Covered topics
- Local images with relative paths  
- URL-based images  
- Images with title attributes  
- HTML `<img>` tag for resizing and advanced formatting  

## Files
- `example.md` — contains image examples  
- `task.py` — summary of the work completed  
- `README.md` — explanation of the task
