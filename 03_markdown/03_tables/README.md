# Markdown — Tables

## Purpose
This task introduces the Markdown syntax for building tables.  
Tables are heavily used in documentation, PBEP structures, task planning,
and technical specifications.

## Covered topics
- Basic tables with headers and rows
- Header separator syntax (`---`)
- Column alignment using colons (`:`)
- PBEP-style structured tables (Step / Action / Description)
