# Markdown — Links & References

## Purpose
This task demonstrates all essential types of hyperlinks used in Markdown.
Links are a fundamental part of documentation, allowing navigation inside
documents, between files, and to external resources.

## Covered topics
- Basic URL links  
- Links with title tooltips  
- Reference-style links  
- Internal anchors  
- Links to local repository files  
- Email links (`mailto:`)  
- Raw GitHub download URLs  

## Files
- `example.md` — demonstration of all link types  
- `task.py` — summary of the work completed  
- `README.md` — explanation of the task
