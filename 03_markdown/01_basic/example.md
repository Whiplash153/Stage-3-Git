# Markdown Basics

## Headings
Markdown uses the '#' symbol for headings.

# H1
## H2
### H3

---

## Text formatting
- *Italic* — wrap text in '*'
- **Bold** — wrap text in '**'
- ***Bold italic*** — combine both

---

## Lists

### Unordered list:
- Item one
- Item two
- Item three

### Ordered list:
1. First
2. Second
3. Third

---

## Links
Here is a link: [OpenAI](https://openai.com)

---

## Horizontal rule
Use three dashes:

---

## Inline code
Use backticks: `print("hello")`
