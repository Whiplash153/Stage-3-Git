# Markdown Basics

## Purpose
This task introduces the essential Markdown syntax used in technical
documentation, README files, and project descriptions.

## Covered topics
- Headings (`#`, `##`, `###`)
- Text formatting (italic, bold, bold italic)
- Unordered and ordered lists
- Hyperlinks
- Horizontal rules
- Inline code using backticks

## Files
- `example.md` — practice file with Markdown examples
- `task.py` — summary of the completed task
- `README.md` — explanation of the task
